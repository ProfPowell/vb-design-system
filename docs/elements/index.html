Elements

Eight design-system documentation components. Each page has a live, themeable example and its full API.

  • <type-specimen>

  • <spacing-specimen>

  • <token-specimen>

    Motion / easing specimen.

    Render a component across variants.

  • <theme-export>

    Export the active theme (DTCG / CSS).

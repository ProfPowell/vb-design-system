    Layout Split

    Main Content (2fr)

    The split layout divides space between two children using a ratio grid. This side takes two-thirds of the available width.

    Sidebar (1fr)

    This side takes one-third. The layout stacks vertically on narrow viewports.

    Layout Switcher

    The switcher layout switches from horizontal to vertical when the container width falls below a threshold.

    Column A

    Each column has equal width above the threshold.

    Column B

    Below the threshold, columns stack vertically.

    Column C

    No media queries needed — pure CSS.

    Layout Prose

    Readable Content Width

    The prose layout constrains content to a comfortable reading width (65ch by default). This ensures that lines of text do not stretch too wide, which would make them harder to read. The optimal measure for body text sits between 55 and 75 characters per line.

    Layout Media Object

    Jane Doe

    The media object layout places a small figure (avatar, icon, thumbnail) beside descriptive content. A classic pattern for comments, notifications, and user cards.

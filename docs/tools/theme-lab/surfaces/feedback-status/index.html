Dialog Variants

Size Variants

Small Dialog

A compact dialog for simple confirmations. Width: 24rem.

Default Dialog

Standard dialog width: 32rem. Suitable for most confirmations and short forms.

Large Dialog

A wider dialog at 48rem. Useful for complex forms, data tables, or detailed content that needs more horizontal space.

Full Dialog

A full-width dialog that stretches to the viewport edge. Used for immersive experiences like media viewers or full-page editors.

Drawer Variants

Right Drawer

Slides in from the right edge. Common for settings panels, detail views, and navigation.

Left Drawer

Slides in from the left edge. Used for primary navigation or sidebars on mobile.

Bottom Sheet

A mobile-style bottom sheet that slides up from the bottom edge. Ideal for action menus and confirmation flows on touch devices.

Top Drawer

Slides down from the top edge. Useful for announcements or notification panels.

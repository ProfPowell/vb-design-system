Feature Cards

  • HTML-First

    Build interfaces with semantic HTML elements instead of div soup. Native elements provide built-in accessibility and browser optimizations for free.

  • Progressive Enhancement

    Every component works without JavaScript. Interactive features layer on top of a solid HTML foundation, ensuring your content is always reachable.

  • Zero Dependencies

    No build tools, no frameworks, no package managers required. Drop a single stylesheet into your project and start building.

  • OKLCH Theming

    Perceptually uniform color spaces produce consistent, vibrant palettes. Switch between dozens of themes with a single attribute change.

  • Layout Attributes

    Declarative layout primitives like stack, grid, and cluster replace bespoke CSS. Compose them freely without writing a single line of custom styling.

  • Accessible

    Semantic markup and ARIA patterns are baked in. Keyboard navigation, screen reader support, and focus management work out of the box.

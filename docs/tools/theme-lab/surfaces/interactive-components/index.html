What is Vanilla Breeze?

A semantic HTML design system that uses CSS custom properties and native elements to create beautiful, accessible interfaces without JavaScript frameworks.

How do themes work?

Themes override CSS custom properties at the root level. Apply a theme by setting the data-theme attribute on the html element.

Is JavaScript required?

Most features work without JavaScript. Web components provide progressive enhancement for interactive patterns like accordions and tabs.

    中文内容 (Chinese)

    什么是 Vanilla Breeze？

    Vanilla Breeze 是一个语义化 HTML 设计系统，使用 CSS 自定义属性和原生元素创建美观、无障碍的界面，无需 JavaScript 框架。

    主题是如何工作的？

    主题通过在根元素级别覆盖 CSS 自定义属性来实现。只需在 html 元素上设置 data-theme 属性即可应用主题。

    是否需要 JavaScript？

    大多数功能无需 JavaScript 即可工作。Web 组件为手风琴和标签等交互模式提供渐进增强。

    Vanilla Breeze क्या है?

    Vanilla Breeze एक सिमेंटिक HTML डिज़ाइन सिस्टम है जो CSS कस्टम प्रॉपर्टीज़ और नेटिव एलिमेंट्स का उपयोग करके सुंदर, सुलभ इंटरफेस बनाता है — बिना किसी JavaScript फ्रेमवर्क के।

    थीम कैसे काम करती हैं?

    थीम रूट स्तर पर CSS कस्टम प्रॉपर्टीज़ को ओवरराइड करती हैं। html एलिमेंट पर data-theme एट्रिब्यूट सेट करके थीम लागू करें।
